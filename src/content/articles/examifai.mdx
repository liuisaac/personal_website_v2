## Project Overview 📋

Examif.AI is an AI-powered web application designed to make learning computer science more accessible and personalized for students at the University of British Columbia. Inspired by the diverse student body and the challenges of grasping technical concepts, the app transforms course content slides into interactive quizzes using ChatGPT, facilitated by AgentHub_’s (now Gumloop's) integration technology.

## Tech Stack 🛠️

Built using React.js, Tailwind CSS, and a Python Flask backend, Examif.AI offers an intuitive and adaptable learning experience for users with varying backgrounds and learning styles. Key challenges included backend-frontend communication and effective prompt engineering for generating high-quality quiz questions.
