## Project Overview 🎮

This Python-based automation bot was designed to dominate the Bomb Party word game. It uses Tesseract OCR to detect the given letter set on screen and leverages Pynput to simulate keystrokes, instantly typing valid words from a preloaded dictionary to win rounds effortlessly.

## How It Works ⚙️

The bot captures the game screen, extracts the prompt using OCR, and searches a dictionary for the first matching word. Once found, it simulates keystrokes using Pynput to type the answer within milliseconds—essentially automating and optimizing human reflexes.
